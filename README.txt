# ClubesLab — Landing
Landing mínima para clubeslab.com.

## Cómo usar
1) Sube estos archivos a un nuevo repositorio en GitHub (p. ej. `clubeslab-landing`).
2) En Vercel, **New Project** → importa el repo → Framework: `Other` (estático).
3) Deploy. Luego en **Domains** añade `clubeslab.com` apuntando los DNS al valor de Vercel.
4) Para capturar emails, edita `index.html` y pon tu `EMAIL_ENDPOINT` (Formspree/MailerLite/Supabase).

### Archivos
- `index.html` — página única.
- `logo.png` — tu logo.
